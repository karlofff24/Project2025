# MyApp
Приложение для управления задачами.
## Установка
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/user/myapp.git
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
## Использование
Запустите приложение:
```bash
npm start
```
